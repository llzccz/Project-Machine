# Java学习路线
## 基础
- 变量、数据类型、运算符、控制语句
- 数组、字符串、基本输入输出流
- OOP基础、封装、继承、多态

## 面向对象
- 设计模式:工厂模式、单例模式、代理模式等
- 接口、抽象类、内部类、枚举类等高级特性
- 异常处理、IO流、反射机制等

## 网络编程
- Socket编程、客户端、服务端
- TCP、UDP协议、Http协议
- HTTP、HTTPs、FTP协议

## 多线程
- 线程的基本概念、创建线程的方法
- 线程同步、原子类、线程池
- 死锁、守护线程、ThreadLocal等

## MySQL数据库
- MySQL基础知识，SQL语句的编写
- MySQL高级特性，存储过程、游标等
- 数据库连接池、数据源等

## Java Web
- Servlet的基本概念、常用接口
- JSP的基本语法、表达式、指令
- MVC模式、RESTful API

## 前端基础
- HTML、CSS、JavaScript基础知识
- DOM操作、jQuery框架
- AJAX、Vue.js等前端框架

## Spring
- IoC、DI、AOP等基础概念
- Spring Bean创建、使用、Bean的作用域
- AOP的基本实现方式、Spring AOP的实现

## SpringMVC
- 基本概念、DispatcherServlet
- 控制器、处理请求、响应试图
- 参数绑定、数据格式化、拦截器等高级特性

## Spring Boot
- 基础概念、系统配置
- 简化Spring开发、自动化配置
- 日志、访问控制、测试等高级特性
